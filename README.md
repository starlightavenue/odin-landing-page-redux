# Odin Landing Page, Redux
This project is a landing page for Starlight Lounge, the spiritual successor to [Starlight Bakery](https://github.com/starlightavenue/odin-landing-page). Starlight Lounge is a cozy spot for dreamers to rest, recharge, and have a tasty treat. It is said that The Lounge reveals itself those who need it most, but no one with pure intentions 
is ever turned away. Many a manifestation has been cultivated within these walls, and many more are sure to follow. Whether it's through a cup of tea or a fresh croissant, I hope the stars align for every soul that crosses the threshold. ✨
